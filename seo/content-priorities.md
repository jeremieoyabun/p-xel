# Content Priorities - P-XEL Studio

## Prioritization Logic

Pages ranked by: conversion potential x search volume x strategic importance.
- Conversion potential: can this page directly generate a booked call?
- Search volume: is there real search demand for the primary keyword?
- Strategic importance: does this page support the overall SEO architecture?

## Content Rules (from CLAUDE.md)

Every page must start with:
- Search intent
- Primary keyword
- Secondary keywords
- Outline

Before writing, verify:
- [ ] Primary keyword is unique (no cannibalization with other pages)
- [ ] Intent matches page type (commercial -> service/city, informational -> blog)
- [ ] Internal links planned (which pages link here, where does this page link)
- [ ] CTA defined (which CTA type: booking, form, case study)
- [ ] Schema identified (which types apply)
- [ ] Word count target set
- [ ] FAQ questions drafted (if applicable)
