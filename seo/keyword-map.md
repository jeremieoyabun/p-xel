# Keyword Map - P-XEL Studio

- One primary keyword per page
- No cannibalization: each keyword is assigned to exactly one page
- Intent drives page type: transactional/commercial -> service/city pages, informational -> blog
- Focus on queries that can lead to booked calls
- Belgian French market: fr-BE terminology and search patterns

## Keyword Clusters (Cannibalization Prevention)

### Cluster: "creation site web"
- Homepage: `studio digital Liege` (broad brand query)
- Service page: `creation site web Belgique` (national service)
- City Liege: `agence web Liege` (local brand)
- City Bruxelles: `agence digitale Bruxelles` (local brand)
- Blog: `combien coute un site web` (informational, price-focused)
- No overlap: each targets a different intent layer

### Cluster: "UX UI design"
- Service page: `UX UI designer Belgique` (national service)
- Blog trends: `tendances UX UI 2026` (informational, trends)
- Blog design system: `design system` (informational, specific topic)
- Blog IA: `IA pour designers` (informational, AI angle)
- Case study Greenmood: `plateforme B2B design` (project-specific)
- No overlap: service page owns the commercial query, blog owns subtopics

### Cluster: "branding"
- Service page: `branding Belgique` (national service)
- Case study PCF: `branding plateforme crowdfunding` (project-specific)
- Case study Kokoro: `branding centre bien-etre` (project-specific)
- No overlap: service page is the pillar, cases are supporting evidence

### Cluster: "local Liege"
- Homepage: `studio digital Liege` (brand + location)
- City page: `agence web Liege` (generic local query)
- Service + city (Phase 2): `creation site web Liege` (specific local service)
- No overlap: different specificity levels

## Metadata Templates

### Service page title
`[Service] | P-XEL Studio - [City or Belgique]`
Example: `Creation Site Web | P-XEL Studio - Belgique`

### Service page meta description
`[Value prop in 1 sentence]. [Deliverables]. [CTA]. Studio digital a Liege.`
Example: `Sites web performants, landing pages et e-commerce. UX/UI, developpement et SEO inclus. Reservez un appel. Studio digital a Liege.`

### City page title
`[Service category] a [Ville] | P-XEL Studio`
Example: `Agence Web a Liege | P-XEL Studio`

### City page meta description
`Studio digital a [Ville]. Sites web, applications, branding et UX/UI. [Differentiator]. Reservez un appel.`

### Blog post title
`[Keyword-rich title] | P-XEL Studio`
Example: `Combien coute un site web en 2026 ? Guide complet | P-XEL Studio`

### Case study title
`[Project Name] - [Project Type] | P-XEL Studio`
Example: `Greenmood - Plateforme B2B Design | P-XEL Studio`
