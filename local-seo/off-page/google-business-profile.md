## Multi-Location Considerations

Current setup: single location in Liege.

**If expanding to Brussels or other cities:**

1. Do NOT create a second GBP listing unless there is a real physical presence (office, coworking membership with mail)
2. Instead, add cities to the service area of the existing Liege listing
3. If a physical Brussels presence is established:
   - Create a second GBP listing: "P-XEL Studio Bruxelles"
   - Same categories, similar description localized to Brussels
   - Unique phone number per location
   - Separate landing page on p-xel.be for each city
4. Never use virtual office addresses - Google penalizes this
5. Each location needs its own unique photos and content
