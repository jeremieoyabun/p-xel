# Citation Building Strategy - P-XEL Studio

## NAP Format (Exact Match Required)

Every citation must use this exact format. Any variation hurts consistency signals.

```
Business name: P-XEL Studio
Address: [Exact street address], [Postal code] Liège, Belgium
Phone: +32 [number]
Website: https://p-xel.be
```

**Rules:**
- Always "Liège" with accent, never "Liege" or "LIEGE"
- Always "P-XEL Studio", never "PXEL", "P-Xel", "Pxel Studio"
- Always the same phone number format with +32 country code
- Always https, never http
- No trailing slash inconsistencies (pick one format and stick to it)

## Belgian-Specific vs International Strategy

### Belgian directories - why they matter more
- Belgian directories send strong local relevance signals to Google
- Pages d'Or is the dominant Belgian directory - a listing here is high value
- 1890.be is the official Walloon business portal - free and authoritative
- CCI and UWE listings build local business credibility and may generate referral traffic
- Sortlist is heavily used by Belgian companies looking for agencies

### International directories - why they still matter
- Clutch, Awwwards, Dribbble build authority and trust signals
- They also generate referral traffic from potential clients doing research
- Clutch reviews specifically influence buying decisions in B2B
- These profiles rank for branded searches ("P-XEL Studio reviews", "P-XEL Studio portfolio")

### Balance
- Prioritize Belgian directories for local SEO impact
- Use international directories for authority, credibility, and portfolio visibility
- Never skip Belgian directories in favor of international ones

## Profile Optimization per Citation

For every listing, not just NAP but also:

1. **Description:** use a localized version of the GBP description, adapted to character limits
2. **Categories:** select the closest match to Web Designer, UX Designer, Graphic Designer
3. **Services:** list all services where the platform allows it
4. **Photos:** upload 3-5 project photos and logo
5. **Links:** always link to https://p-xel.be (homepage) and Calendly
6. **Social links:** add where allowed (LinkedIn, Dribbble, Behance)

## Monitoring Strategy

### Monthly checks
- Use a spreadsheet to track all 30 citations: platform, URL, status (submitted, live, needs update)
- Check that NAP is still correct on each platform (some directories edit or truncate data)
- Check for duplicate listings on Google, Pages d'Or, Yelp, Foursquare - merge or remove duplicates

### Tools
- **BrightLocal** or **Whitespark** for automated citation monitoring (optional, paid)
- **Manual audit** quarterly: search "P-XEL Studio" on Google and check first 3 pages for incorrect listings
- **Google Alerts** for "P-XEL Studio" to catch new mentions

### Fix issues immediately
- Wrong phone number on any platform: fix within 24h
- Wrong address format: fix within 24h
- Duplicate listing found: request removal or merge same day
- NAP consistency is the single most important factor in citation value
