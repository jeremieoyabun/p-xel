# Local Landing Page Skill

## Goal
Build city or location pages that are useful, specific, and locally relevant.

## Required output
- primary keyword
- location intent
- city intro
- service relevance in that location
- local trust angle
- FAQs
- CTA
- internal links
- schema suggestions

## Rules
- no spun city pages
- add meaningful local context
- avoid duplicate patterns
- make each page genuinely useful
