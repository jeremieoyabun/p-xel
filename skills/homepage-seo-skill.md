# Homepage SEO Skill

## Goal
Write a homepage that is premium, clear, and conversion-focused while still supporting SEO.

## Workflow
1. Define intent
2. Define primary keyword
3. Define secondary keywords
4. Build outline
5. Write hero
6. Write credibility
7. Write services
8. Write process
9. Write proof
10. Write CTA
11. Check keyword placement
12. Check clarity and conversion

## Rules
- premium tone
- no fluff
- no vague claims
- clear business value
- must feel stronger than typical agency sites
