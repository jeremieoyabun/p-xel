# Service Page Skill

## Goal
Create a service page that ranks and converts.

## Required output
- SEO title
- meta description
- H1
- intro
- service explanation
- deliverables
- who it is for
- process
- FAQ
- CTA
- internal links
- schema recommendations

## Rules
- one page, one core intent
- explain the service in concrete language
- connect strategy, execution, and results
